注: 请分别对每个查询字段的值做 encodeURIComponent> 转义处理后再拼装Url

    例:
    redirect=https://www.baidu.com/
    转为
    redirect=https%3A%2F%2Fwww.baidu.com%2F

Get usermap

参数 值 必填 说明
method getusermap 是 调用方法
redirect https://www.baidu.com/ 是 重定向地址 
    https://joehecn.github.io/sso/#method=getusermap&redirect=https%3A%2F%2Fwww.baidu.com%2F

Set usermap

参数 值 必填 说明
method setusermap 是 调用方法
username joe 是 用户名
uuid xxxx 否 凭证，为空时表示 logout 删除用户
redirect https://www.baidu.com/ 是 重定向地址 
    # 添加用户
    https://joehecn.github.io/sso/#method=setusermap&username=joe&uuid=xxxx&redirect=https%3A%2F%2Fwww.baidu.com%2F

    # 删除用户
    https://joehecn.github.io/sso/#method=setusermap&username=joe&uuid=xxxx&redirect=https%3A%2F%2Fwww.baidu.com%2F
